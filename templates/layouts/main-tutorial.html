Creating Answers

Multiple Choice Answers

For multiple choice answers, simply write an answer, select its validity, write the reason why it is true or false, select an image if you so desire and submit! (Images are optional for multiple choice answers.)

Short Answers

For short answers, simply write an answer, the reason why it is true or false and submit!

Point and Click Answers/Image Clicking Answers

You will be asked to designate an area on the question image as either a right or wrong answer. Be sure to pick a size for the box and be careful not not overlap answers. Students will not be able to see the box.

Drag and Drop Answers

First, Pick an image and decide whether this answer will be correct or incorrect.\n Next, you will be asked to pick a size for the answer image and to place this image on the question image where you would like the student to drag their answer.\n Only right answers will be given a dropbox.
