Professor Content

Create Content

Creating Classes

On the professor homepage, find the "Add Class" button on the left hand side and enter a class name and a class code. Class codes should be kept reasonably private, and only be distributed to your class. Students will need to be provided with the code to access the class.

Creating Chapters

To create a chapter, on the edit class page click the "Add Chapter" button on the upper left. You will be presented with a dialogue and asked to enter a chapter name and a chapter number. The number is the ordering index of the chapter, ie. where you want the chapter to be relative to other chapters. Try to avoid redundant chapter numbers.

Creating Sections

To create a section, on the edit chapter page click the "Add Section" button on the upper left. You will be presented with a dialogue and asked to enter a chapter name and a chapter number. The number is the ordering index of the section, ie. where you want the sections to be relative to other sections. Try to avoid redundant section numbers.

Creating Questions

To create a question, find the "Add Question" button on the edit section page. You will be presented with a dialogue that will prompt you for an order number, the question text itself, the question type and an image upload. Please note that you must add an image to drag and drop/point and click questions.

Creating Answers

Multiple Choice Answers

For multiple choice answers, simply write an answer, select its validity, write the reason why it is true or false, select an image if you so desire and submit! (Images are optional for multiple choice answers.)

Short Answers

For short answers, simply write an answer, the reason why it is true or false and submit!

Point and Click Answers/Image Clicking Answers

You will be asked to designate an area on the question image as either a right or wrong answer. Be sure to pick a size for the box and be careful not not overlap answers. Students will not be able to see the box.

Drag and Drop Answers

First, Pick an image and decide whether this answer will be correct or incorrect. Next, you will be asked to pick a size for the answer image and to place this image on the question image where you would like the student to drag their answer. Only right answers will be given a dropbox.

Creating Glossary Terms

To start, navigate to the edit class page. Below the list of chapters you should find an "Open Glossary" button. Once you navigate to the edit glossary page, note the upper left click the button "Add Term". Once you click this button you will be presented with a dialogue asking for a term and its definition. Once your ready to go hit submit and your new term is live!

Edit Content

Editing Classes

To edit a class, navigate to the professor home and click the value you would like to change (ClassName or ClassCode) and enter a value you would prefer. Then click the Save button to the right. If you would like to make changes to the class' chapter, sections or glossary click the Make Edits/Additions button to the left of the class name on the professor home page.

Editing Chapters

To edit a chapter, navigate to the edit class page and click the value you would like to change (Chapter Number or Chapter Name) and enter a value you would prefer. Then click the Save button to the right. If you would like to make changes to the chapter's sections, click the Add/Edit Sections button to the left of the chapter number on the edit class page.

Editing Sections

To edit a section, navigate to the edit chapter page and click the value you would like to change (Section Number or Section Name) and enter a value you would prefer. Then click the Save button to the right. If you would like to make changes to the chapter's sections, click the Add/Edit Section Content button to the left of the section number on the edit chapter page.

Editing Questions

To edit a question, navigate to the edit section page and click the value you would like to change (Question Number, Question Name or Question Type) and select a value you would prefer. Then click the Save button to the right. If you would like to make changes to the question's answers, click the Add/Edit Answers button to the left of the question number on the edit section page.

Editing Answers

To edit an answer, navigate to the edit question page and click the value you would like to change (Answer Text, Answer Correctness or Answer Reason) and select a value you would prefer. Then click the Save button to the right. You can also change images associated with the answer to the right of the Save button. You must delete and recreate drag and drop/point and click answers you to make changes to their location, size, color or image.
